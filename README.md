# Project 1: Binary Classification of Human Risk Decision-Making Behavior in Simulated Game Scenarios

**Student ID:** 414551032  

This project focuses on building a **Binary Classification of Human Risk Decision-Making Behavior in Simulated Game Scenarios** to analyze human risk decision-making behavior in game scenarios.

The dataset simulates how players make decisions under **high-risk and high-reward situations**, such as whether to challenge a difficult enemy, take risks with limited resources, or choose between defensive and aggressive strategies under time pressure.
